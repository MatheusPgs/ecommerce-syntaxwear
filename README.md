# SyntaxWear - Tênis e Sneakers Online

Este é o repositório do projeto SyntaxWear, uma landing page para um e-commerce de tênis e sneakers. O site foi desenvolvido com foco em responsividade e design moderno, apresentando uma estrutura clara e fácil de navegar.

## Funcionalidades

- **Header Responsivo:** Navegação principal e ícones de acesso rápido (conta, ajuda, carrinho).
- **Seção Hero:** Banner de destaque com chamada para ação para explorar modelos e comprar.
- **Categorias de Produtos:** Cards visuais para as categorias de tênis (Casual, Esporte, Moderno, Futurista).
- **Grid de Produtos:** Layout para exibição de produtos em destaque e variados.
- **Footer Completo:** Inclui formulário de inscrição para newsletter, links para redes sociais e navegação secundária por categorias.

## Tecnologias Utilizadas

- **HTML5:** Estrutura semântica do conteúdo.
- **CSS3:** Estilização completa, incluindo variáveis CSS, reset de estilos e modularização por componentes para facilitar a manutenção e responsividade.
  - **Fontes:** Utiliza a fonte 'Ubuntu' do Google Fonts.

## Como Visualizar o Projeto

1.  **Clone o repositório:**
    ```bash
    git clone https://github.com/MatheusPgs/ecommerce-syntaxwear.git
    ```
    (Nota: Substitua `https://github.com/MatheusPgs/ecommerce-syntaxwear.git` pelo URL real do seu repositório, se aplicável.)
2.  **Navegue até o diretório do projeto:**
    ```bash
    cd ecommerce-syntaxwear
    ```
3.  **Abra o arquivo `index.html` em seu navegador preferido.**
